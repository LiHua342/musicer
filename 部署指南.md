# 🚀 Musicer 部署指南

你的音乐播放器是纯静态项目（HTML + CSS + JS），无需服务器，部署非常简单。以下是 **4 种免费方案**，按推荐度排序。

---

## 方案一：GitHub Pages ⭐ 推荐

**优点**：完全免费、稳定、绑定自定义域名、自动 HTTPS

### 步骤

**1. 创建 GitHub 仓库**

访问 [github.com](https://github.com) 注册/登录，点击右上角 `+` → `New repository`。

- Repository name: `musicer`（或任意名称）
- 选择 **Public**（公开）
- 不要勾选任何初始化选项

**2. 推送代码到 GitHub**

```bash
# 在项目根目录执行
git init
git add .
git commit -m "初始化音乐播放器项目"
git branch -M main
git remote add origin https://github.com/你的用户名/musicer.git
git push -u origin main
```

**3. 启用 GitHub Pages**

- 进入仓库 → `Settings` → `Pages`
- Source 选择 `Deploy from a branch`
- Branch 选择 `main`，文件夹选择 `/ (root)`
- 点击 `Save`

等待 1-2 分钟，你的网站将部署在：

```
https://你的用户名.github.io/musicer/
```

---

## 方案二：Vercel 🚀 最快

**优点**：零配置、自动部署、全球 CDN 加速

### 步骤

**1. 访问 [vercel.com](https://vercel.com)**

使用 GitHub 账号登录。

**2. 导入项目**

- 点击 `New Project`
- 选择你的 GitHub 仓库 `musicer`
- 无需任何配置，直接点击 `Deploy`

30 秒内完成部署，你会得到一个：

```
https://musicer-xxx.vercel.app
```

每次 `git push` 后 Vercel 会自动重新部署。

---

## 方案三：Netlify 🎯 最简单

**优点**：拖拽部署、无需命令行

### 步骤

**1. 访问 [netlify.com](https://netlify.com)**

使用 GitHub 账号登录。

**2. 拖拽部署（零命令行）**

- 进入 `Sites` 页面
- 直接将你的 `musicer` 项目文件夹**拖拽**到浏览器窗口中
- 等待上传完成，网站即可访问

你会得到一个：

```
https://随机名称.netlify.app
```

可以在设置中修改为自定义子域名。

---

## 方案四：Cloudflare Pages 🌍

**优点**：全球最强 CDN、无限带宽

### 步骤

**1. 访问 [dash.cloudflare.com](https://dash.cloudflare.com)**

注册/登录 Cloudflare 账号。

**2. 进入 Workers & Pages → Create → Pages → Connect to Git**

- 选择你的 GitHub 仓库
- Build settings 留空（无需构建）
- 点击 `Save and Deploy`

---

## 方案对比

| 方案 | 免费额度 | 自定义域名 | 自动部署 | 学习成本 |
|------|---------|-----------|---------|---------|
| GitHub Pages | 无限流量、100GB 空间 | ✅ | ❌ 需手动 | ⭐⭐ |
| Vercel | 100GB/月 带宽 | ✅ | ✅ Git Push | ⭐ |
| Netlify | 100GB/月 带宽 | ✅ | ✅ Git Push / 拖拽 | ⭐（最低） |
| Cloudflare Pages | 无限带宽 | ✅ | ✅ Git Push | ⭐⭐ |

---

## 快速起步建议

- **不熟悉 Git** → 用 **Netlify 拖拽部署**，1 分钟搞定
- **熟悉 Git** → 用 **Vercel**，体验最好
- **想要最专业** → 用 **GitHub Pages**，生态最完整

## 补充：本地预览

部署前可以在本地预览效果：

```bash
# 方式一：Python（Windows 自带或需安装）
python -m http.server 8080

# 方式二：npx（需安装 Node.js）
npx serve .
```

然后在浏览器打开 `http://localhost:8080`。
